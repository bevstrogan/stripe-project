Данный проект представляет собой небольшое приложение магазина с возможностью создавать платежные формы через библиотеку Stripe.
Для удобства была создана небольшая верстка на базе шаблона с Bootstrap. 

У приложения реализован следующий функционал:
* Просмотр всех товаров на главной странице
* Просмотр детальной информациии о конкретном товаре
* Создание Stripe сесиии для оплаты конкретного товара
* Admin функционал для работы с моделью

Для запуска программы необходимо:
* Создать и заполнить файл .env переменными окружения по примеру с .env.sample
* Запустить сервер путем написания в консоль комманды: python manage.py run server
